# 工作邮件提示词

## 邮件回复助手

### 用途
帮助快速回复各种工作邮件，保持专业且友好的语调。

### 提示词
```
你是一个专业的邮件回复助手。请根据以下要求帮我写邮件回复：

1. 保持专业且友好的语调
2. 回复要简洁明了
3. 根据邮件内容调整回复的正式程度
4. 如果需要，包含下一步行动建议

原邮件内容：[在此粘贴原邮件]

回复要点：[说明需要回复的主要内容]
```

### 示例
**输入**：客户询问产品功能的邮件
**输出**：专业详细的功能说明回复

## 会议邀请邮件

### 用途
快速生成会议邀请邮件模板。

### 提示词
```
请帮我写一封会议邀请邮件，包含以下信息：

会议主题：[会议主题]
会议时间：[具体时间]
会议地点/链接：[地点或在线会议链接]
参会人员：[参会人员列表]
会议议程：[主要讨论内容]

要求：
- 语调专业友好
- 格式清晰易读
- 包含必要的会议准备事项
```

## 项目进度汇报

### 用途
向上级汇报项目进展情况。

### 提示词
```
请帮我写一份项目进度汇报邮件，包含：

项目名称：[项目名称]
汇报时间：[本周/本月]
已完成工作：
- [完成事项1]
- [完成事项2]

进行中工作：
- [进行中事项1]
- [进行中事项2]

遇到的问题：[如有问题请列出]
下周计划：[下周工作安排]

要求简洁明了，突出重点，体现工作成果。
```
