#### 2. **Variable Aggregator详细说明（新增内容）**

```markdown
## Variable Aggregator（变量聚合器）

### 核心功能
- **多分支聚合**：将问题分类或条件分支的多路输出聚合为单路
- **工作原理**：按参数顺序选择输出，输出第一个非空的变量值

### 应用场景
1. **问题分类后的多路聚合**
   - 将question-classifier的多个输出路径聚合
   - 统一后续节点的输入处理

2. **IF/ELSE条件分支后的聚合**
   - 合并不同条件分支的结果
   - 简化工作流连接复杂度

3. **多数据源整合**
   - 聚合来自不同来源的变量
   - 实现数据的统一处理

### 配置示例
```yaml
- data:
    type: variable-aggregator
    title: 结果聚合
    variables:
      - variable: aggregated_result
        sources:
          - "{{#classification_result#}}"
          - "{{#fallback_result#}}"
```

### 官方文档
- [Variable Aggregator官方文档](https://docs.dify.ai/guides/workflow/node/variable-aggregator)
```

#### 3. **Agent节点详细说明（新增内容）**

```markdown
## Agent节点（自主推理）

### 核心概念
工作流中的"大脑"，让LLM自主决策和处理任务，具备：
- **自主推理能力**：根据输入自动选择最佳处理路径
- **工具调用能力**：智能使用各种工具完成复杂任务
- **上下文理解**：维护对话状态和任务进度

### Agent策略
可插拔的推理算法模块，定义LLM如何思考和使用工具：

#### ReAct策略
- **思考-行动-观察**的经典链式推理
- 适用于需要逐步分析的复杂任务
- 提供清晰的推理过程追踪

#### Function Calling策略
- 精确的基于函数的调用方式
- 适用于结构化任务处理
- 高效的工具集成能力

### 使用场景
1. **智能客服**：自动理解用户问题并调用相应工具
2. **数据分析**：根据数据特征选择合适的分析方法
3. **内容生成**：智能组合多种生成工具创建内容

### 官方文档
- [Agent节点介绍](https://docs.dify.ai/guides/workflow/node/agent)
```

#### 4. **并行处理功能（新增内容）**

```markdown
## 工作流并行处理

Dify v0.8.0引入的强大并行处理能力，支持四种并行模式：

### 1. 简单并行
- **特点**：从固定节点创建多个并行分支
- **应用**：相似子任务的并发处理
- **示例**：多语言翻译、多模型比较

### 2. 嵌套并行
- **特点**：多级并行结构
- **结构**：
  - 第一级：主要分支分离
  - 第二级：每个分支内的并行处理
- **应用**：复杂任务的层次化处理

### 3. 迭代并行
- **特点**：循环结构内的并行处理
- **应用**：批量数据处理场景
- **优势**：显著提升大规模数据处理效率

### 4. 条件并行
- **特点**：基于条件运行不同的并行任务分支
- **应用**：根据状态和需求灵活执行任务
- **优势**：智能的资源分配和任务调度

### 官方文档
- [工作流并行处理](https://docs.dify.ai/guides/workflow/parallel-processing)
```

#### 5. **文件上传功能（新增内容）**

```markdown
## 文件上传功能

### 支持格式
- **文档文件**：PDF, DOC, TXT等文本文档
- **音频文件**：MP3, WAV等音频格式
- **视频文件**：MP4, AVI等视频格式

### 处理方式
1. **文档文件处理**
   - 使用Doc Extractor节点提取文本内容
   - 支持OCR文字识别
   - 保持文档结构和格式

2. **音频/视频处理**
   - 需要配置音频转文本工具
   - 支持视频关键帧提取
   - 多模态内容分析

### 应用场景
- **文档问答**：上传文档进行智能问答
- **报告摘要**：自动生成文档摘要
- **电子表格处理**：数据分析和处理
- **多媒体分析**：音视频内容理解

### 配置注意事项
- 文件类型与变量名匹配：`video_files` 配套 `type: files`
- 设置合理的文件大小限制
- 配置正确的文件扩展名白名单

### 官方文档
- [工作流文件上传功能](https://dify.ai/blog/introducing-dify-workflow-file-upload-a-demo-on-ai-podcast)
```
