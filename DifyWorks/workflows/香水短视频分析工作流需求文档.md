# 香水短视频分析工作流需求文档

## 📋 **项目概述**

### 🎯 **核心目标**
构建一个专业的香水类短视频内容分析与优化工作流，面向西班牙TikTok市场，提供从内容识别到策略优化的全链路分析服务。

### 🎭 **智能体角色定义**

**名称**: 短视频精灵 (Perfume Video Wizard)

**专业领域**:
- 🎬 短视频内容分析与审核
- 🌹 香水产品营销策略  
- 🇪🇸 西班牙市场本地化
- 📱 TikTok平台内容优化

**核心能力**:
1. **多模态内容识别**: 视频、音频、文字全维度解析
2. **专业审核评估**: 基于7大维度的科学评价体系
3. **本地化策略**: 西班牙文化与语言双重适配
4. **创意优化建议**: 具体可执行的改进方案

---

## 🔄 **工作流程设计**

### 📥 **输入阶段**
- **主要输入**: 香水类短视频文件
- **支持格式**: MP4, MOV, AVI等主流视频格式
- **文件限制**: 建议15-60秒，文件大小不超过100MB

### ⚙️ **处理流程**

#### 第一步：多维度内容识别
```yaml
任务: 全面解析视频内容
技术要求:
  - 视频帧分析: 关键帧提取、画面元素识别
  - 音频处理: 语音转文字、背景音乐识别  
  - 文字识别: 字幕、屏幕文字、品牌标识
  - 场景理解: 拍摄环境、道具布置、光线效果
输出: 结构化的内容识别报告
```

#### 第二步：专业维度评估
```yaml
任务: 基于7大专业维度进行量化评分
评估维度:
  1. 吸引力指数 (0-100分)
  2. 创意融合度 (0-100分)  
  3. 主题聚焦度 (0-100分)
  4. 节奏控制力 (0-100分)
  5. 文案引导力 (0-100分)
  6. 画面美感度 (0-100分)
  7. 声音配乐度 (0-100分)
输出: 详细的评分报告和问题诊断
```

#### 第三步：本地化策略生成
```yaml
任务: 生成西班牙市场适配方案
要求:
  - 文化适配: 考虑西班牙用户偏好和文化禁忌
  - 语言优化: 中文-西班牙语双语文案
  - 平台特性: TikTok算法偏好和传播逻辑
输出: 3条本地化文案 + 策略建议
```

#### 第四步：优化建议输出
```yaml
任务: 提供可执行的改进方案
内容包括:
  - 即时优化: 可立即调整的问题点
  - 重拍建议: 需要重新制作的核心要素
  - 策略升级: 长期的内容策略改进
输出: 结构化的优化行动清单
```

## 📊 **评估标准体系**

### 1. 🎯 **吸引力维度** (前3秒黄金法则)

#### 📈 **评分细则**
- **90-100分**: 多重吸引元素完美融合，瞬间抓住注意力
- **80-89分**: 有明显亮点，能有效吸引用户停留
- **70-79分**: 有吸引元素但不够突出
- **60-69分**: 吸引力一般，需要优化
- **60分以下**: 缺乏有效吸引点

#### 🔍 **具体评估指标**

**A. 情绪冲击力 (40%权重)**
```yaml
评估要点:
  - 情绪钩子强度: 惊喜/好奇/共鸣/冲突
  - 开场节奏: 是否在前3秒建立情绪连接
  - 真实性: 避免过度表演和虚假情绪
评分标准:
  - 优秀: 自然真实的情绪表达，引发强烈共鸣
  - 良好: 有情绪点但略显刻意
  - 一般: 情绪表达平淡
  - 较差: 虚假做作，引起反感
常见问题:
  - ❌ "震惊！这款香水让男神为我疯狂" (过度夸张)
  - ✅ "用了三年的香水，今天才发现这个秘密" (自然好奇)
```

**B. 视觉张力 (35%权重)**
```yaml
评估要点:
  - 色彩搭配: 是否有视觉冲击力
  - 构图美感: 画面是否有层次和重点
  - 动作设计: 是否有吸引眼球的动作或手势
香水特色要求:
  - 瓶身展示角度和光线运用
  - 喷洒瞬间的美感捕捉
  - 香水与人物/场景的和谐融合
评分标准:
  - 优秀: 视觉元素丰富且和谐，有明显美感
  - 良好: 有视觉亮点但整体感一般
  - 一般: 画面普通，缺乏特色
  - 较差: 视觉杂乱或过于单调
```

**C. 卖点切入速度 (25%权重)**
```yaml
评估要点:
  - 产品出现时机: 是否在3秒内展示产品
  - 卖点表达清晰度: 核心价值是否快速传达
  - 引导明确性: 是否明确指向产品优势
香水产品特殊性:
  - 香调描述的感官表达
  - 使用场景的快速建立
  - 品牌调性的视觉体现
```

  - 约会前的仪式感
  - 工作日的自信加成
  - 特殊场合的气场营造

**B. 创意与产品融合度 (35%权重)**

  - 自然融入: 创意是否服务于产品展示
  - 记忆关联: 用户是否能记住产品特点
