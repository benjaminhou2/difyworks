## 📥 **主分析Prompt**

### 🔍 **内容识别与评估阶段**

```markdown
请对上传的香水短视频进行全面分析，按照以下步骤执行：

## 第一步：视频内容识别

**技术解析要求**：
1. **基本信息提取**：视频时长、分辨率、主要场景
2. **画面内容描述**：
   - 拍摄场景和环境布置
   - 香水产品展示方式和角度
   - 人物动作和手势(考虑无人脸出镜限制)
   - 道具使用和色彩搭配
3. **文字信息识别**：字幕、屏幕文字、品牌标识
4. **音频内容分析**：口播内容、背景音乐风格、音效使用
5. **产品信息识别**：香水品牌、瓶身设计特征、使用场景

## 第二步：7维度专业评估

按照以下评估体系对视频进行量化评分(每个维度100分，总分700分)：

### 1. 🎯 吸引力维度 (前3秒表现)
**评分权重分配**：
- 情绪冲击力(40分)：开场3秒内的情绪钩子强度
- 视觉张力(35分)：色彩、构图、动作的视觉吸引力
- 卖点切入(25分)：产品和核心价值的展示速度

**评分标准**：
- 90-100分：多重吸引元素完美融合，瞬间抓住注意力
- 80-89分：有明显亮点，能有效吸引用户停留
- 70-79分：有吸引元素但不够突出
- 60-69分：吸引力一般，需要优化
- 60分以下：缺乏有效吸引点

**特别关注**：
- 避免过度夸张："震惊！这款香水让男神为我疯狂"
- 推荐自然好奇："用了三年的香水，今天才发现这个秘密"

### 2. 🎨 内容创意维度
**评分权重分配**：
- 共鸣感建立(40分)：是否贴近西班牙女性日常生活
- 创意融合度(35分)：创意与产品展示的自然结合
- 场景代入感(25分)：使用场景的真实性和代入感

**西班牙市场特殊要求**：
- 生活方式是否贴近当地女性日常
- 情感表达是否符合拉丁文化习惯
- 香水使用场景是否符合当地社交文化

### 3. 🎯 主题聚焦维度
**评分权重分配**：
- 卖点明确度(45分)：核心价值表达的清晰度
- 场景契合度(35分)：使用场景与产品特点的匹配
- 用户匹配度(20分)：内容与目标用户群体的契合

**香水卖点层次分析**：
- 基础功能：香味特征、持久度、适用场合
- 情感价值：魅力提升、自信建立、个性表达
- 社交价值：印象改善、社交加分、品味体现

### 4. ⏰ 节奏控制维度
**评分权重分配**：
- 切换顺畅度(40分)：画面转换的自然流畅程度
- 高潮设置(35分)：是否有明显的情绪高潮点
- 音画同步度(25分)：BGM与画面节奏的协调性

**香水拍摄特殊要求**：
- 喷洒动作的慢镜头处理效果
- 瓶身细节的特写时机
- 香味扩散的视觉化表达

### 5. 📝 文案引导力维度
**评分权重分配**：
- 钩子文案效果(45分)：开场文案的吸引力和自然度
- 卖点导入逻辑(35分)：从钩子到产品的逻辑流畅性
- 互动引导自然度(20分)：评论引导的自然程度

**避免模板化表达**：
- ❌ "99%的女生都不知道..."
- ❌ "看完这个你就明白了..."
- ✅ 创造独特且自然的表达方式

### 6. 🎨 画面美感维度
**评分权重分配**：
- 技术质量(40分)：清晰度、稳定性、曝光控制
- 美学风格(35分)：色彩搭配与香水调性的匹配
- 产品展示质量(25分)：香水瓶身的展示效果

**无人脸拍摄创意评估**：
- 手部动作的优雅程度
- 身体局部展示的美感
- 环境氛围的营造效果

### 7. 🎵 声音配乐维度
**评分权重分配**：
- 口播质量(45分)：语音清晰度和情绪表达
- 配乐契合度(35分)：BGM与香水调性、西班牙文化的匹配
- 整体协调性(20分)：音画同步和音效运用

## 第三步：问题诊断
基于评估结果，识别主要问题：
- 列出得分低于80分的维度
- 分析具体问题点和影响程度
- 评估改进的紧急程度和难易度

**输出要求**：
每个维度提供具体评分和详细评分依据，问题诊断要具体到可操作的改进点。
```

## 🚀 **优化建议生成Prompt**

```markdown
基于前述分析结果，提供具体可执行的优化建议：

## 建议分类体系

### 即时优化 (可立即调整，无需重拍)
针对评分60-79分的问题，提供：
- **具体问题点**：明确指出需要调整的内容
- **调整方法**：详细的操作步骤

- 发布策略(时间、标签、封面)

### 重拍建议 (需要重新制作内容)
针对评分60分以下的严重问题，提供：
- **核心问题分析**：根本原因和影响程度
- **重拍关键要点**：必须改进的核心元素
- **参考案例方向**：成功案例的特点分析

**重点关注**：
- 开场3秒的吸引力设计

基于整体表现，提供：

- 透明瓶身的反光控制技巧
```

## 🔧 **Dify工作流配置建议**

### 📋 **推荐节点流程**

1. **Start节点**
   ```yaml
   配置：
     - 文件类型：files
     - 变量名：video_files
     - 最大长度：48
     - 允许格式：mp4,mov,avi
   ```

2. **Doc-Extractor节点**
   ```yaml
   功能：提取视频中的文字信息
   输入：{{#start.video_files#}}
   输出：extracted_text
   ```

3. **LLM节点 (主分析)**
   ```yaml
   模型：gpt-4o (支持视频分析)
   温度：0.3
   输入：
     - 视频文件：{{#start.video_files#}}
     - 提取文字：{{#doc_extractor.extracted_text#}}
   Prompt：[使用上述主分析Prompt]
   ```

4. **Variable-Aggregator节点**
   ```yaml
   功能：聚合分析结果
   输入：
     - 分析结果：{{#llm_analysis.text#}}
     - 评分数据：{{#llm_analysis.structured_output#}}
   ```

5. **LLM节点 (文案生成)**
   ```yaml
   模型：gpt-4o
   温度：0.7
   输入：{{#variable_aggregator.output#}}
   Prompt：[使用上述文案生成Prompt]
   ```

6. **Template节点**
   ```yaml
   功能：格式化最终报告
   模板：[使用上述输出格式模板]
   ```

7. **End节点**
   ```yaml
   输出：完整的分析报告
   ```

### ⚙️ **关键配置要点**

- **视频分析能力**：确保使用支持视频处理的模型（如gpt-4o）
- **多语言支持**：西班牙语生成需要专门的语言模型配置
- **评分系统**：使用结构化输出确保评分的一致性
- **模板格式**：保证输出格式的规范性和可读性

这套Prompt模板可以直接应用于Dify工作流，为香水短视频分析提供专业、系统、可操作的解决方案。
